# tmux

- <c-space> = <prefix>
- <prefix> : = command prompt
- <prefix> r = reload the configuration

## tmux-copycat

- <prefix> / = search in the shell output
- <prefix> <M-h> = jump over SHA-1/SHA-256 hashes
- <prefix> <C-u> = url search

## extrakto

- <prefix> <tab> = fuzzy search in the shell output
